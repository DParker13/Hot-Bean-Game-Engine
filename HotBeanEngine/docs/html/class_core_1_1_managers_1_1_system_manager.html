void Core::Managers::SystemManager::EntitySignatureChanged ( Entity entity,
Signature entity_signature )

Notifies each system that an entity's signature has changed.

Parameters
entityThe ID of the entity whose signature has changed.
entity_signatureThe new signature of the entity.

This function iterates over each system and checks if the entity's new signature matches the system's signature. If it does, the entity is added to the system's set of entities. If it does not, the entity is removed from the system's set of entities.
